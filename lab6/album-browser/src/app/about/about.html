About This Project

Meow Album Browser is a laboratory project built with Angular. It uses the public JSONPlaceholder API to simulate working with real data.

Main features:

  • Fetching albums from API
  • Viewing album details
  • Editing album titles (PUT request)
  • Deleting albums (DELETE request)
  • Viewing album photos

This project demonstrates basic frontend architecture, routing, services, and working with REST APIs.
